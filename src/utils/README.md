# Utils

This directory contains utility functions and helpers.

## Planned Utilities

- `unitConversions.ts` - Unit conversion functions (yards/meters, MIL/MOA, etc.)
- `validation.ts` - Data validation helpers
- `formatting.ts` - Number and date formatting
- `calculations.ts` - Common calculation helpers
- `storage.ts` - AsyncStorage helpers
- `permissions.ts` - Permission request helpers
