# Screens

This directory contains all screen components for the Mobile DOPE app.

## Planned Screens

- **Dashboard** - Main overview screen
- **RifleProfiles** - List and manage rifle profiles
- **RifleProfileDetail** - View/edit individual rifle profile
- **AmmoProfiles** - List and manage ammunition profiles
- **AmmoProfileDetail** - View/edit individual ammo profile
- **BallisticCalculator** - Calculate ballistic solutions
- **DOPELog** - View and manage DOPE logs
- **DOPELogEntry** - Create/edit DOPE log entries
- **RangeSession** - Fast-entry mode for field use
- **TargetMarking** - Mark POI on target images
- **DOPECardGenerator** - Generate printable DOPE cards
- **Settings** - App settings and preferences
