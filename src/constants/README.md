# Constants

This directory contains application constants.

## Planned Constants

- `colors.ts` - Color palette for dark/light themes
- `typography.ts` - Font sizes and styles
- `sizes.ts` - Touch target sizes, spacing, etc.
- `calibers.ts` - Common caliber database
- `reticles.ts` - Reticle type database
- `manufacturers.ts` - Optic and ammo manufacturers
