# Models

This directory contains data models and their validation logic.

## Planned Models

- `RifleProfile.ts` - Rifle profile data model
- `AmmoProfile.ts` - Ammunition profile data model
- `EnvironmentSnapshot.ts` - Environmental conditions model
- `DOPELog.ts` - DOPE log entry model
- `ShotString.ts` - Chronograph shot string model
- `RangeSession.ts` - Range session model
- `TargetImage.ts` - Target image with POI markers model
- `AppSettings.ts` - Application settings model

Each model includes:
- TypeScript interface/class definition
- Validation rules
- Serialization methods (toJSON/fromJSON)
- Factory methods for testing
